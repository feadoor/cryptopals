Creates ECB-encrypted user tokens.

A black box which takes an email address, sanitises the input, creates a token of the form email=<user_email>&uid=10&role=user, then encrypts that under ECB using a block size of 16 bytes, and returns the result.

Goal

To obtain (by any means) a token which decrypts to one containing "role=admin".
